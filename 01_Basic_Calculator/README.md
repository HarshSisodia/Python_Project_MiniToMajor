# Basic Calculator

This is my first Python mini project created while learning Python. The main purpose of this project was to practice functions, conditional statements, and loops by building a simple command-line calculator.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Average of two numbers
- Modulus
- Handles division by zero
- Handles invalid menu selection
- Continue or exit option after every calculation

## Concepts Used

- Functions
- If-Else Statements
- While Loop
- User Input
- Arithmetic Operators

## How to Run

1. Clone this repository.

```bash
git clone https://github.com/HarshSisodia/Python_Project_MiniToMajor.git
```

2. Open the project folder.

```bash
cd Python_Project_MiniToMajor/01_Basic_Calculator
```

3. Run the program.

```bash
python Calculator.py
```
