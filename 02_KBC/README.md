# KBC Quiz Game

This is a simple KBC-style quiz game that I created while learning Python. I made this project to practice the Python concepts I have learned so far and understand how they can be used together in a small project.

## Features

- Multiple-choice questions
- Prize money for every correct answer
- 50-50 lifeline
- The 50-50 lifeline can only be used once
- Option to quit the game anytime
- Game ends if the answer is wrong
- Shows the final prize money at the end

## Concepts Used

- Functions
- Lists
- For Loop
- While Loop
- If-Elif-Else Statements
- Boolean Variables
- User Input
- String Methods
- Return Statement
